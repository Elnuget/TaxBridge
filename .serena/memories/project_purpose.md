# Propósito del Proyecto TaxBridge

TaxBridge es un sistema integral de gestión tributaria desarrollado con enfoque principal en el frontend/maquetación usando Angular, y backend en Node.js/Express con MongoDB.

## Objetivos Principales
- Desarrollar una maqueta funcional y profesional del frontend
- Implementar componentes visuales completos con diseño responsive
- Crear una experiencia de usuario (UX/UI) completa para presentación al cliente
- Integrar funcionalidades básicas del backend para persistencia de datos

## Enfoque Actual
- **Frontend Prioritario**: Maquetación, componentes, diseño y UX/UI
- **Backend Integración**: Creación automática de clientes, sistema de cuentas, checkout, dashboard personalizado
- **No se enfoca**: Funcionalidades complejas de backend, solo integración necesaria

## Funcionalidades Clave
- Flujo de compra completo sin registro previo
- Creación automática de cuentas de cliente con contraseñas temporales
- Dashboard personalizado mostrando productos comprados
- Sistema de autenticación básico
- Dashboards separados para admin, contador y cliente

## Tech Stack
- **Frontend**: Angular 20, CoreUI, Bootstrap
- **Backend**: Node.js, Express, MongoDB
- **Estilos**: SCSS, Prettier para formato
- **Testing**: Jest para backend, Angular testing para frontend
